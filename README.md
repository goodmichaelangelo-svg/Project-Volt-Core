# Project Volt: The Internet of Electricity ⚡

Project Volt is an AI-driven, decentralized peer-to-peer (P2P) energy trading protocol built to solve the global power crisis. 

By automating the flow of energy between localized solar networks and idle private battery storages, Project Volt builds a virtual power plant grid controlled entirely through a mobile interface.

## 🛠️ Current Features (v1.0 Core Backend)
- **Automated P2P Handshake**: Intelligently routes electrical current based on real-time neighborhood battery percentages.
- **Fintech Wallet Core**: Implements sub-cent automated billing transaction fees ($0.01 per kW traded) to generate platform utility revenue.
- **Interactive CLI Simulator**: A real-time command-line interface simulating the consumer dashboard experience.
